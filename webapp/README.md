# BAMF Quiz Web Application

This directory contains the production web application files for the German citizenship test quiz.

## Files Overview

### Core Application
- **`index.html`** - Main HTML page with disclaimer and quiz interface
- **`app.js`** - Quiz application logic and functionality  
- **`styles.css`** - Application styling and responsive design
- **`sw.js`** - Service worker for offline support and caching

### Data
- **`bamf-questions-merged.json`** - Bilingual question database (German/English)
  - Contains all 310 official BAMF citizenship test questions
  - Generated by the infrastructure translation tools

## Deployment

This is a **static web application** that can be deployed to any web server or CDN:

### Simple Deployment Options
```bash
# Python built-in server (for testing)
python3 -m http.server 3000

# Node.js serve
npx serve .

# Any static hosting service:
# - GitHub Pages
# - Netlify
# - Vercel
# - AWS S3
# - Apache/Nginx
```

### Production Deployment
1. Copy all files in this directory to your web server
2. Ensure the server serves static files with appropriate MIME types
3. Configure HTTPS for security (recommended)
4. Set up appropriate caching headers for the JSON data file

## Features

- ✅ **Complete Question Set**: All 310 official BAMF questions
- ✅ **Bilingual Support**: German originals + English translations  
- ✅ **Multiple Modes**: Quiz, Study, and Answers-only modes
- ✅ **Progress Tracking**: Local storage saves your progress
- ✅ **Responsive Design**: Works on desktop and mobile
- ✅ **Offline Support**: Service worker enables offline usage
- ✅ **Issue Reporting**: Integrated GitHub issue creation for feedback

## Legal Notice

This is an **unofficial** practice tool and is **not affiliated** with BAMF or the German government. Always verify answers with official BAMF sources before your actual test.

## Data Updates

To update the question data, use the infrastructure tools in the `../infrastructure/` directory to re-extract and translate questions, then replace `bamf-questions-merged.json`.
